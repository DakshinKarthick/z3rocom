# Developer Setup — Z3roCom

Generated: 2025-11-11

## Prerequisites
- Node.js 18+ and npm/pnpm
- Docker (optional for local services)

## Installing dependencies
1. Frontend (client):
   - cd client
   - npm install

2. Backend (server):
   - cd server
   - npm install

## Running the app locally
- Start backend: cd server; npm start
- Start frontend: cd client; npm run dev

## Notes
- package.json files are empty; populate them with dependencies and scripts before running
